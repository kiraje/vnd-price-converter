Summary

VND Price Converter does not collect, store, or transmit any personal data. Your browsing activity stays entirely on your device.

Data Collection

This extension collects no user data whatsoever. Specifically:

  • No personal information is collected
  • No browsing history is tracked or stored
  • No analytics or telemetry data is gathered
  • No cookies are set by the extension
  • No data is sold or shared with third parties

Network Requests

The extension makes a single type of network request: fetching the current USD to VND exchange rate from the Frankfurter API (a free, open-source API powered by the European Central Bank). This request:

  • Contains no user information
  • Contains no browsing data
  • Is a simple GET request to api.frankfurter.app/latest?from=USD&to=VND
  • Occurs once per hour while Chrome is running

Local Storage

The extension stores only two things locally on your device using Chrome's storage API:

  • The cached exchange rate (a single number)
  • Your on/off preference

This data never leaves your device.

Permissions

  • activeTab: Required to read page content and detect USD prices for conversion
  • storage: Required to cache the exchange rate and save your preferences
  • alarms: Required to schedule hourly exchange rate updates

This extension is open source. You can review the complete source code at: github.com/kiraje/vnd-price-converter
